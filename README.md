# Quiz Web – HTML / CSS / JavaScript

## Description
Ce projet est un site web simple qui permet de tester les connaissances en
HTML, CSS et JavaScript à travers un quiz à choix multiples.

Le site a été réalisé uniquement avec :
- HTML
- CSS
- JavaScript
## Fonctionnement
- L’utilisateur répond aux questions du quiz.
- Chaque question possède une seule bonne réponse.
- En cliquant sur le bouton « Valider », le score final est affiché.
